Lead Generation Demo – BD Web Agent (PoC)

This is a small demo that shows how a business development–focused tool
can find, enrich, and rank potential leads for 3D in-vitro models.
What this demo includes
-Lead data loaded from a CSV file (no live APIs)
-Simple rule-based scoring to rank leads by likelihood
-A Streamlit app to view and sort the ranked leads
-A mix of real public PubMed data and synthetic example profiles

How to run
pip install streamlit pandas
streamlit run app.py
